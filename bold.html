商品

商品的含义

用于交换的劳动产品

知识产权是一种知识商品，也具有价值和使用价值

商品的基本属性

  • 价值

  • 使用价值

商品的基本属性 使用价值 价值
含义 满足某种需要 无差别的人类劳动
主体 买者占有 卖者拥有
性质 自然属性 社会 / 本质属性
推导关系 商品→使用价值；价值→使用价值 商品→价值；价值→商品
联系 使用价值是价值的物质承担者 商品是使用价值和价值的统一 / （周测9/29）消费者关注食品安全；优质优价、物美价廉、食品安全

货币

货币的含义

货币是从商品中分离出来固定地充当一般等价物的商品

与一般等价物的区别在于「固定」
- 比特币是虚拟商品而不是法定货币

货币的本质

一般等价物

货币的产生

货币是商品交换发展到一定程度的的产物

最初的商品交换是物物交换

货币的基本职能

  1. 价值尺度

    只需要观念上的货币

    价格是价值的货币表现

  2. 流通手段

    首付、邮购

    商品交换 > 商品流通 - 商品—货币—商品 / 从商品到货币的转化

    流通中所需要的货币量 = 商品价格总额 / 货币流通速度

    商品价格总额 ≠ 价值总额

  3. 贮藏手段

    中国大妈买黄金

  4. 支付手段

    支付宝、碌卡、赊账买卖、工资、税、利息、还款、地租（非买卖）

  5. 世界货币
  • 支付宝不是「纸币」，不能「代替执行」货币的职能，更不等于「本身具有」货币的职能
  • 不只货币能承担价值尺度职能，一切商品都能承担价值尺度职能

纸币

纸币的含义

由国家（或某些地区）发行的、强制使用的价值符号

国家有权发行纸币，不可以任意发行纸币，纸币的发行量必须以流通中所需要的货币量为限度。

国家有权决定纸币的面值及每一种面值的量，但不能决定纸币的购买力

纸币发行 过多 / 通货膨胀 过少 / 通货紧缩
社会供求 供不应求 供过于求
结果 人民生活水平下降，经济稳定受影响 生产积极性降低，生产缩小，人民生活整体经济萧条

  • 失业率与货币工资变化率 / 通货膨胀率 / 物价上涨率呈负相关

  • 经济增长率 / 货币工资变化率与通货膨胀率 / 物价上涨率 呈正相关

外汇与汇率

外汇

外汇不仅仅包括外币，还包括用外币表示的票据、股票和债券等

  • 我国的外汇是指以美元表示的用于国际间结算的支付手段

汇率变化与币值的关系

1
本币价值 外币价值 / 外币外汇汇率 本币汇率 本币 外币汇率 / 本币外汇汇率 外币
上升 下降 上升 升值 下降 贬值

美元汇率 = 美元对人民币的汇率

人民币外汇汇率 ≠ 人民币 / 100外币

  • 人民币升值不利于物价稳定，但不至于企业破产
    • （2013茂名月考）降低外汇储备贬值风险可以增加大宗商品和战略性材料储备，推进人民币国际化进程→人民币成为区域货币

汇率变化对国家经济的影响

本币升值

  • 出口减少，进口增加；
  • 外资投资成本增加；
  • 导致国内生产下降、失业增加；
  • 利于本国居民出国学习、旅游；
  • 本国外汇储备贬值；
  • 有利于减轻外债负担（不是减少外债数额）

本币贬值

  • 出口增加，进口减少；
  • 外资投资成本降低；
  • 增强本国产品的国际价格竞争力，有利于本国企业走出去；
  • 有利于劳务输出；
  • 本国消费相对便宜，外国游客增加；
  • 增加外债负担

保持人民币币值基本稳定的意义

美国迫使人民币升值主要是为了削弱中国出口商品的竞争力 ，其实质就是要抑制中国的发展。

一国的汇率是由一国的经济决定的，汇率的变动也是由经济的综合情况来决定的。

人民币坚持以市场供求为基础、有管理的浮动汇率制度。2013年12月中央经济工作会议对2014年人民币汇率走势提出了明确要求，即“保持人民币汇率基本稳定”。有利于人民币国际化的稳步推进，世界金融的稳定和经济的发展。
