Features

Fundamental features

  • GitHub Flavored Markdown (GFM)
  • Live preview with scroll sync
  • Fenced code blocks with syntax highlight
  • Tables & Alignment
  • Export to PDF & HTML
  • Drag & drop to open
  • Convenient toolbar icons

Extra features

  • Task lists
  • Emojis & Font icons
  • Mathematical formulae
  • Flowcharts
  • Sequence diagrams
  • Gantt diagrams
  • Vim mode
